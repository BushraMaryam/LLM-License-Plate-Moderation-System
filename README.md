# LLM-License-Plate-Categorization-Evaluation-Framework-In-Databricks
A comprehensive evaluation pipeline for LLM-based license plate classification. Implements multiple reasoning and validation strategies:

Zero-shot inference

True NLI (Entailment/Contradiction)

Rule-based consistency metrics

Toxicity scoring

Optional language detection

Confusion matrices and summary reports
Use Case: Model evaluation, AI fairness assessment, and categorisation reliability analysis.
